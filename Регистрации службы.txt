Регистрация приложения в качестве службы с помощью NSSM:
Загрузите NSSM:
Скачайте NSSM с официального сайта: https://nssm.cc/download.

Добавьте NSSM в системную переменную PATH:
Разархивируйте скачанный файл, найдите исполняемый файл nssm.exe и добавьте его путь в системную переменную PATH. Это позволит вам запускать nssm из любого места в командной строке.

Зарегистрируйте службу с помощью NSSM:
Откройте командную строку и выполните следующую команду, заменив YourServiceName, Путь\К\Вашему\Приложению.exe  на соответствующие значения:
nssm install YourServiceName "Путь\К\Вашему\Приложению.exe"

Затем настройте вашу службу с помощью команды:


nssm set YourServiceName AppDirectory "Путь\К\Вашему\Приложению"

nssm set YourServiceName AppStdout "Путь\К\Вашему\Лог\Файлу.log" 

Запустите службу командой:

nssm start YourServiceName
